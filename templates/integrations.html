How to get your Airtable API key:

  1. Go to Airtable API tokens
  2. Click "Create new token"
  3. Name your token (e.g., "Workflow Automation")
  4. Select the appropriate scopes for your bases
  5. Copy the generated API key

How to get your Notion API token:

  1. Go to Notion Integrations
  2. Click "New integration"
  3. Name your integration and select the workspace
  4. Click "Submit"
  5. Copy the "Internal Integration Token"
  6. Important: You need to share each database with your integration
